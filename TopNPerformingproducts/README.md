# Top N Product Analysis

## Objective
Visualize the top-performing products dynamically, allowing users to select Top 3, Top 5, or Top 10 products using a slicer.
## Dataset
The dataset has 3 columns: Product, Category, and Sales.

## Key Features
- Dynamic Top N selection using a slicer (Top 3, Top 5, or Top 10)
- Ranking with `RANKX` and DAX measures  
- Conditional formatting to highlight top products in blue
- Dynamic subtitle showing total sales and % contribution  
- Tooltips displaying product rank, category, and products count
